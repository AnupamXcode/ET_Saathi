## 🧠 CORE FEATURES

### 1. 🔐 Authentication System

* Login / Signup (JWT-based authentication)
* Secure session handling
* Store users in database

---

### 2. 👤 User Profile Engine

Store:

* Portfolio (stocks like Reliance, HDFC, Infosys)
* Risk level (Low / Medium / High)
* Investment goal (optional)

Allow:

* Edit profile
* Portfolio visualization (charts)

---

### 3. 📰 News Intelligence Engine

Input:

* Financial news (text)

AI must:

* Extract key event (rate hike, merger, crash)
* Identify affected sectors
* Perform sentiment analysis
* Output structured JSON

---

### 4. 🔮 What-If Scenario Engine (CORE WOW FEATURE)

Input examples:

* "What if Reliance falls 10%?"
* "What if interest rates increase by 1%?"

System must:

* Simulate ripple effects across sectors
* Identify impacted companies
* Predict market movement (Nifty / sectors)
* Provide actionable insights

---

### 5. 📊 Decision Engine

Based on:

* News + User Profile + Scenario

Output:

* Recommendation → Buy / Hold / Avoid
* Confidence Score (0–100%)
* Risk Level

---

### 6. 💸 Simulation Engine

Input:

* Investment amount (₹X)

Output:

* Estimated future value
* Comparison:

  * Invest vs Not Invest
* Use realistic financial assumptions (no fake predictions)

---

### 7. ⚠️ Risk & Validation Engine

Detect:

* Unrealistic claims
* Misleading news

Output:

* Warning messages
* Confidence adjustments

---

### 8. 🧠 Explainability Engine

ALWAYS explain:

* Why recommendation was made
* In simple, beginner-friendly language
